\# FocusMate Project Instructions



This is a React Native + Expo + TypeScript mobile app.



Project goal:

FocusMate is an AI companion-based planner app for task, date, and meeting management. The mascot is Milo, a cute caring green dinosaur companion.



Tech stack:

\- Expo

\- React Native

\- TypeScript

\- React Navigation

\- AsyncStorage

\- expo-notifications

\- expo-speech

\- expo-haptics

\- @react-native-community/datetimepicker



Important rules:

\- Do not remove existing screens unless asked.

\- Do not change navigation route names unless necessary.

\- Keep UI consistent with src/theme.ts.

\- Use Milo assets from assets/mascot.

\- Do not put OpenAI API keys, WhatsApp tokens, or Supabase service keys inside the mobile app.

\- Backend integrations should be prepared as separate services later.

\- After changes, run TypeScript or Expo checks when possible.



Useful commands:

\- npx expo start -c

\- npx tsc --noEmit
